# Postman Collection for Workout Tracker API

This folder contains Postman collections and environments for testing the Workout Tracker API.

## Collections

- `workout_tracker.postman_collection.json` - Complete API collection

## Environments

- `local.postman_environment.json` - Local development environment

## Usage

1. Import the collection and environment into Postman
2. Select the local environment
3. Run the requests to test the API endpoints
